# Agent 工作流自动化使用场景

English version: [Use Cases for Agent Workflow Automation](Use-Cases-Agent-Workflow-Automation.md)

clawvisual 适合 URL 转轮播图与 agent 自动化场景。

## 1. 创作者内容复用

- 输入：newsletter/文章 URL
- 输出：标题、caption、hashtags、slide 文案和生成图片
- 价值：一篇长文快速拆成多平台传播素材

## 2. 增长团队内容流水线

- 输入：活动文档或产品更新说明
- 输出：带品牌风格的轮播草稿与修订结果
- 价值：提升社媒内容产出效率与一致性

## 3. 多 agent 系统中的 MCP 工具

- 输入：来自编排 agent 的结构化任务
- 输出：异步 job id 与可轮询结果
- 价值：作为可复用技能接入更大自动化流程

## 4. 编辑审校 + 修订闭环

- 输入：已生成结果 + 审校指令
- 输出：文案改写或封面/指定页重生成
- 价值：无需重跑全流程即可做人机协同优化

## 5. 私有化部署自动化

- 输入：内部知识文档或链接
- 输出：可视化传播内容
- 价值：数据留在私有环境，仍可集成 API/MCP
