# Use Cases for Agent Workflow Automation

中文版本: [Agent 工作流自动化使用场景](Use-Cases-Agent-Workflow-Automation.zh-CN.md)

clawvisual is designed for URL-to-carousel workflows and agent automation.

## 1. Creator content repurposing

- Input: newsletter/article URL
- Output: post title, caption, hashtags, slide copy, and generated images
- Benefit: turn one long piece into multi-platform social assets

## 2. Growth team content pipeline

- Input: campaign doc or product update
- Output: branded carousel drafts with revision loops
- Benefit: faster social publishing with consistent format

## 3. MCP tool in multi-agent systems

- Input: structured tasks from orchestrator agents
- Output: async job IDs and result payloads
- Benefit: clawvisual can run as a reusable skill in larger agent workflows

## 4. Editorial review + revision loop

- Input: generated result plus reviewer instruction
- Output: revised copy style or regenerated cover/slides
- Benefit: human-in-the-loop quality control without restarting from scratch

## 5. Internal self-hosted automation

- Input: internal docs/knowledge pages
- Output: visual-ready social summaries
- Benefit: private deployment with API or MCP integration
